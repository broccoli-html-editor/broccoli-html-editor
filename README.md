# broccoli

## Usage

### NodeJS

```js
var Broccoli = require('broccoli');
var broccoli = new Broccoli({
	'Mod1': '/path/to/moduleset1/',
	'Mod2': '/path/to/moduleset2/',
	...
});

```

### Browser JS

```html
<!-- broccoli -->
<script src="client/dist/broccoli.min.js"></script>
```
